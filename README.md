# PersonaRewards – Reimagining Digital Advertising Value

## 🎯 Overview
PersonaRewards is a browser extension that enables users to engage with web content more meaningfully and get rewarded for their attention. While digital advertising has historically overlooked the end-user, PersonaRewards introduces a more balanced value exchange by allowing users to earn rewards based on their interactions and interests — all while retaining full control over their personal data.

## ✨ Core Features

- 🔍 **Interest-Based Engagement**: Analyzes web activity (with user consent) to identify content categories and interests.

- 🎁 **Reward System**: Users earn tokens and digital collectibles for participating in surveys, feedback, and quests.

- 🛡️ **Privacy-Centric Design**: Users manage their own interest profiles with transparency and control over what's stored and shared.

- 🧩 **Gamification Layer**: Progress through tiers and unlock unique digital badges for achievements and participation.

## 🛠️ Technology Stack

- **Frontend**: React, TailwindCSS, Framer Motion
- **Extension Framework**: Chrome APIs, Node.js
- **Blockchain**: Ethereum-compatible testnets
- **Storage**: Local browser storage & decentralized metadata storage
- **AI Integration**: Used for intelligent content engagement and interest tagging

## 🚀 Getting Started

### Requirements
- Node.js v16+ (v16.15.0 recommended)
- Chrome Browser (v100+)
- A crypto wallet extension (e.g., MetaMask v10.0+)

### Setup Instructions

```bash
git clone https://github.com/web2-5-leaders/personarewards
cd personarewards
npm install
npm run build
```

Then go to `chrome://extensions` → Enable Developer Mode → Load the `dist` folder as unpacked.
